    The Problem

    Growing locations means growing complexity

    More venues, more rooms, more scheduling clashes. Without structure, expansion becomes chaos.

    Room double-bookings

    Two teachers scheduled in the same room at the same time. It happens more than you'd like.

    Closure dates are manual

    One venue closed for maintenance, another open as normal. Tracking which closures apply where is error-prone.

    No overview across locations

    You can't see what's happening at each venue without checking multiple spreadsheets or calendars.

    The Solution

    Locations and rooms built into the scheduler

    Every location, room, and availability window is connected to the calendar. The system prevents conflicts automatically.

    Multiple locations

    Add as many teaching venues as you need. Each has its own address, contact details, and settings.

    Room management

    Create rooms within each location. Set capacity, availability windows, and equipment lists.

    Location-specific closures

    Set closure dates per location or across the whole organisation. Lessons skip closures automatically.

    Teacher assignment

    Assign teachers to locations. Their availability is scoped to the locations where they teach.

    Cross-location visibility

    Admins see all locations in one calendar view. Filter by venue, room, or teacher.

    Utilisation tracking

    See how much each room and location is used. Identify spare capacity or bottlenecks.

    Utilisation

    Frequently asked questions

    Can I manage multiple teaching locations?
    Yes. Create as many locations as you need, each with their own address, rooms, availability, and closure dates.
    Can I assign teachers to specific locations?
    Absolutely. Teachers can be assigned to one or more locations. Their availability is set per location.
    Do rooms have their own availability?
    Yes. Each room within a location can have its own availability windows and capacity. The scheduler respects room availability.
    Can I set closure dates per location?
    Yes. Set location-specific closure dates (e.g., building maintenance) or organisation-wide closures (e.g., bank holidays).
    Can I see utilisation per location?
    Yes. Utilisation reports show how much each location and room is being used, helping you optimise space.
